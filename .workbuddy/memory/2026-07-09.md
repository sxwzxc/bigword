## 新增「动效字符」功能（小文字拼大文字 + 动画）

- 抽取共享逻辑到 `src/lib/bigword.ts`：`BUILTIN_FONTS`/`TEXT_COLORS`/`BG_COLORS`、`containsCJK`、`measureCellMetrics`、`toFullwidth`、`rasterizeTarget()`（画布采样返回 darkGrid/cols/rows）、`generateBigWord()`（字符串输出，逻辑与旧版一致）、`buildAnimatedCells()`（返回每格 AnimCell 二维数组，供逐字动画）。

- 新建 `src/app/animate/page.tsx`：客户端页面。控制项：素材文本/目标文本/目标字体/素材字体/字号密度/动画类型（脉冲放大·波浪起伏·纵向滚动·闪烁·彩虹流光·无）/速度/文字色/背景色，含复制、下载、打乱。每个暗格渲染为 `<span>` 套用 CSS 动画，纵向滚动用内联 strip 双拷贝无缝循环。
