## 「动效字符」页面（/animate）最终实现：独立 Canvas 2D + requestAnimationFrame

- **定位**：完全独立实现，NOT import `bigword.ts` 任何东西（用户明确要求不要复用字符画页面的逻辑）。
- **核心引擎** `src/lib/animate-text.ts`：
  - 自带字体表 `ANIMATE_FONTS`（不共享 BUILTIN_FONTS）。
  - `AnimatedBigText` 类：离屏 `<canvas>` 把目标文字画成位图 → `getImageData` 按网格采样 alpha 得到"暗格 mask"（每暗格放一个小文字）。
  - `requestAnimationFrame` 逐帧 `ctx.fillText` 渲染每个暗格小文字，并对每格施加变换/颜色。
  - **随机缩放（默认动画 "随机缩放"）**：在 `compute()` 里给每个格子生成「决定性随机种子」`{phase, spd}`（由 col/row 异或哈希），`drawFrame` 中 `scale = lo + (hi-lo)*(0.5+0.5*sin(t*ω*spd+phase))`，每格以自己的相位+频率独立放大缩小、随时间持续变化。
  - **静态暗底**：先以低透明度（0.14）画一层静止的小字，保证"大形状(B)"始终清晰可读；亮色小字在其上各自跳动，读作"小A住在大B里"。
  - 其他动画：wave（列向波浪起伏）、scroll（素材字在格内纵向流动）、blink（逐格呼吸透明度）、rainbow（逐格 hsl 色相流动）、none。
  - 强度滑块控制 `scale`（hi），`lo = max(0.3, 2-hi)`；速度滑块控制周期；支持 `prefers-reduced-motion`（定格）。
  - `toPNG()` 导出当前帧为图片。
- **页面** `src/app/animate/page.tsx`：canvas 预览 + 控件（素材/目标文本、目标字体、素材字体、密度=列数、动画类型、速度、强度、文字色、背景色、下载图片）。默认演示 source="A"、target="B"、字体 Arial，直接对应"用A组成B、A随机缩放"的需求。
