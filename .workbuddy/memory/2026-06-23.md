## 第十四轮: 半角采样网格彻底修复CJK对齐(核心修复)
- 核心问题: CJK模式采样网格用全角宽,空格半角只占半个cell → 后续全部错位
- 解决: 半角采样网格 + 全角字符占2cell
  1. measureCellMetrics CJK模式 charAspect=2.0(cellH=2×cellW)
  2. generateBigWord CJK模式 cols翻倍
  3. isFullwidthChar()/charCellWidth()/cellStep() 字符宽度分类: fullwidth=2, halfwidth=1
  4. 普通模式: 全角字符填cell[c]+覆盖cell[c+1](空占位''), c+=step; join时空占位占0字符,全角字符自然占2位置
  5. 防截断模式: token.cellLen=词内字符cell宽度之和; 全角字符同样设空占位
  6. PNG导出: maxCols按视觉宽度算, x按charCellWidth推进
- commit a01a163 → origin/main
